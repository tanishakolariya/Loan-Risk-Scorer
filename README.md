 Project Overview

Financial institutions face significant risk when approving loans.
This project builds a Loan Default Prediction System that:

Analyzes applicant financial data

Predicts default probability

Provides real-time results via a web interface

The model is trained using historical loan data and deployed as a Streamlit application.
